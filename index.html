The Coder's Almanac

The language is built on 6 core keywords. Master them, and you master the craft.

Core Keywords

Keyword Purpose Example
wee Variable declaration wee x = 5
woo Print statement woo x
weewoo If statement weewoo (x > 5) { ... }
woowee Else clause woowee { ... }
woowoo Function definition woowoo main() { ... }
weewoowee Return statement weewoowee 42

Variables & Expressions

Declare with wee, assign with =. Standard arithmetic is supported.


# Declare a new variable
wee x = 10

# Re-assign its value
x = x + 5       # (x is now 15)
x = x * 2       # (x is now 30)

# Print the final value
woo x           # Prints 30

Control Flow

Direct your program's path with weewoo (if) and woowee (else).


wee power_level = 9001

weewoo (power_level > 9000) {
    woo "It's over 9000!"
} woowee {
    woo "Just a regular day."
}

Defining Functions

Craft your own reusable logic. This function adds two numbers.


# Define a function to add two numbers
woowoo add(a, b) {
    weewoowee a + b
}

# The main function
woowoo main() {
    wee result = add(33, 44)
    woo result  # Prints 77
    weewoowee 0
}
